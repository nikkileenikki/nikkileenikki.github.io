# Repository File Structure

## Important: Two Separate Applications

This repository contains TWO different applications:

### 1. Creative Support Tools (Root)
- **Location**: `/index.html` (repository root)
- **Purpose**: Creative Support tools page with Ratio Calculator
- **URL**: https://nikkileenikki.github.io/
- **⚠️ CRITICAL**: This file should NEVER be replaced or overwritten

### 2. HTML5 Ad Builder (Subfolder)
- **Location**: `/kult-adbuilder/index.html`
- **Purpose**: HTML5 ad creation and animation tool
- **URL**: https://nikkileenikki.github.io/kult-adbuilder/
- **Build Source**: Generated from `/src/index.tsx`

## Build & Deploy Notes

When building the Ad Builder:
1. Build output goes to `/dist/` folder
2. Static files copied to `/kult-adbuilder/` folder
3. **NEVER** overwrite `/index.html` (root) - it's the Creative Support page
4. Only update `/kult-adbuilder/index.html` for the Ad Builder

## File Preservation Rules

**Always Keep These Files Unchanged:**
- `/index.html` - Creative Support tools (28KB file with Ratio Calculator)
- `/styles/style.css` - Styles for Creative Support tools

**Can Be Updated:**
- `/kult-adbuilder/index.html` - Ad Builder app (auto-generated from localhost:3000)
- `/kult-adbuilder/static/app.js` - Ad Builder JavaScript
- `/kult-adbuilder/static/styles.css` - Ad Builder styles
- `/src/index.tsx` - Ad Builder source (Hono server template)
- `/public/static/*` - Ad Builder source files

## Deployment Commands

```bash
# Build Ad Builder (does NOT affect root index.html)
npm run build

# Copy Ad Builder files to kult-adbuilder folder
cp public/static/app.js kult-adbuilder/static/
curl -s http://localhost:3000 > kult-adbuilder/index.html
sed -i 's|/static/|static/|g' kult-adbuilder/index.html

# Push to GitHub (preserves root index.html)
git push origin main
git push -f pages main
```

## Important Notes for Future Development

- The root `/index.html` is a completely separate application
- Never run commands that overwrite the root index.html
- Always verify file paths when copying/building
- Keep the two applications isolated from each other
